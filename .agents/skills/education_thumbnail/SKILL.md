# Education YouTuber Thumbnail Strategy Module

This module encapsulates the exact thumbnail prompt strategy, design guidelines, and overlay rules tailored for **Education Content Creators** (e.g., Physics Wallah, Vedantu, Unacademy, RGPV, University & Competitive Exam Channels).

---

## 1. Module Overview & Guidelines

When generating thumbnails for Education Creators:
- **Tone & Style**: High-impact, authoritative, urgent, and visually engaging.
- **Reference Aesthetics**: Bold typography, dramatic lighting, expressive presenters, color-coded text banners, and official notice card overlays.
- **Color Rules**: Dominant brand color (e.g. `#1a3fd4` Royal Blue) with high-contrast accent highlight (`#f5d800` Bright Yellow) and pure white text (`#FFFFFF`).

---

## 2. Layout Structure

- **LEFT (40%)**: Subject / Presenter with dramatic rim lighting in brand color. Hands/props must be topic-specific (e.g. holding blueprints/T-squares for Engineering, formula sheets for Maths, printed notice circulars for Exam Updates).
- **CENTER**: Topic visual metaphors (diagrams, technical tools, calendar callout popups).
- **RIGHT & OVERLAYS**: Stacked ALL-CAPS headlines, pill badges, and urgent notice strips.

---

## 3. Preserved Education Prompt System

### A. Topic Visual Context Rule
> "The background scene, any props the subject holds, any visible objects, and the overall visual metaphor must all clearly communicate this specific topic at a glance. For example: if the topic is Engineering Drawing, include drafting tools, T-squares, blueprints, technical sheets. If Maths, include equations on a board. If History, include relevant artefacts."

### B. Overlay Configuration Rules
1. **Top Badge**: Topic pill tag (e.g. `RGPV | MATHS-II`).
2. **Main Headline**: Stacked Impact/condensed font (Line 1: White, Line 2: Yellow/Accent).
3. **Official Alert Card**: White notice card overlay for exam updates, postponements, or results.
4. **Bottom Banner**: Full-width dark strip with yellow accent highlight word.

---

## 4. Usage

This module represents the complete prompt specification for the **Education YouTuber** niche without requiring any modifications to the core application code.
